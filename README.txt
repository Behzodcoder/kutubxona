Для запуска Sass:

Settings -> json -> добавляем эти строки
"liveSassCompile.settings.formats": [
        {
          "format": "expanded",
          "extensionName": ".css",
          "savePath": "css/",
          "savePathReplacementPairs": null
        }
      ],

потом можно нажимать на "Watch Sass",

Иначе в папке "Scss" откроются множества ненужных CSS файлов.
